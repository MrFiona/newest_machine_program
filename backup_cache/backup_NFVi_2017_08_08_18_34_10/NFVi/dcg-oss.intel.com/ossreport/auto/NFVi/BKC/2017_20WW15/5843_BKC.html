Key Updates

\xc2\xb7        Total test cases: 174; Attempted test cases: 174 (Attempted rate: 100%); Passed test cases: 162 and pass rate is 93.10%.\nPlease visit  HPQC for test details


Release Note

\xc2\xb7       \nThe release is based on Ubuntu14.04.5 with kernel 3.13.0-100 generic, only contain the SW package,\nno auto-installer support

\xc2\xb7       \nInstallation step: 

\xc2\xa7  Install\nUbuntu14.04.5 Server

\xc2\xa7  Download\nthe SW package and unzip it, Copy the APP folder to /opt/ of target SUT

\xc2\xa7  Navigate\nto /opt/APP/script, run update_kernel.sh to update kernel to 3.13.0-100, reboot\nthe system 

\xc2\xa7  Navigate\nto /opt/APP/script, run setup.sh to install required libs and SW dependencies
